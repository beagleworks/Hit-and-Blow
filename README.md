# ヒットアンドブロー (Hit and Blow Game)

シンプルで楽しい数字当てゲーム「ヒットアンドブロー」のWebアプリケーション版です。

## ゲーム概要

CPUがランダムに選んだ4桁の数字を推理して当てるゲームです。各予想に対して「Hit（ヒット）」と「Blow（ブロー）」のヒントが与えられます。

## 遊び方

1. ゲームが開始されると、CPUが4桁の数字を秘密裏に選びます
2. プレイヤーは4桁の数字を入力して予想します
3. 予想に対して以下のフィードバックが返されます：
   - **Hit（ヒット）**: 数字と位置が両方正解
   - **Blow（ブロー）**: 数字は含まれているが位置が違う
4. ヒントを手がかりに正解を推理し、4 Hitを目指します

### 例

- 正解が `1234` の場合：
  - `1567` → 1 Hit, 0 Blow（1が正しい位置）
  - `5134` → 2 Hit, 1 Blow（1と3が正しい位置、4は含まれるが位置が違う）
  - `1264` → 2 Hit, 1 Blow（1と4が正しい位置、2は含まれるが位置が違う）
  - `1234` → 4 Hit（正解！）

## ゲーム設定

設定ボタン（⚙️）から以下のオプションを変更できます：

### 1. 数字の重複

- **許可しない（デフォルト）**: 答えに同じ数字が複数回使われることはありません
  - 例: `1234`, `5678`, `0ABC` など
- **許可する**: 答えに同じ数字が複数回使われることがあります
  - 例: `1123`, `9999`, `00AA` など

重複なしモードでは、入力時に同じ数字を重複して入力することはできません。

### 2. 使用する数字の種類数

スライダーで6～16種類の数字を選択できます：

| 種類数 | 使用される文字 | 例 |
|--------|----------------|-----|
| 6 | 0-5 | `0123`, `4502` |
| 10（デフォルト） | 0-9 | `1234`, `9876` |
| 11 | 0-9, A | `12A4`, `A987` |
| 16 | 0-9, A-F | `1A3F`, `BEEF` |

10種類を超える場合、9の次からアルファベット（A-F）が使用されます。

## 機能

- リアルタイムフィードバック（Hit/Blow表示）

## 起動方法

1. リポジトリをクローン：
   ```bash
   git clone https://github.com/beagleworks/Hit-and-Blow.git
   cd Hit-and-Blow
   ```

2. ブラウザで`index.html`を開く、またはローカルサーバーを起動：
   ```bash
   python3 -m http.server 1236
   ```
   uvを使用している場合：
   ```bash
   uv run -m http.server 1236
   ```   
   その後、ブラウザで`http://localhost:1236`にアクセス
